라우터/네트워크 정리 페이지


라우터 명령어

명령어 역할
enable / en 관리자 모드로 --> Router#
configure terminal / conf t 관리자 모드에서 설정모드로 --> Router(config)#
copy r s 라우터 셋팅 내용을 NVRAM에 저장
hostname [이름] 호스트 이름 명명
login 로그인 명령
exit 이전 모드로 돌아감 / 종료
end 관리자 모드로 돌아감
enable password [암호] 관리자모드로 엑세스할 때 암호 설정(암호화되지 않고 저장)
enable secret [암호] 관리자모드로 엑세스할 때 암호 설정(암호화되어 저장)
erase startup-config 라우터 설정 명령을 지우고 초기화
line console 0 콘솔라인 모드 전환 명령
line vty 텔넷 등 가상 터미널 전환 명령


자주 나오는 유형

1) 인터페이스 정보 확인

enable - show interface - copy r s


2) 접속한 사용자 확인

enable - show user(s) - copy r s


3) 라우팅 테이블 확인

enable - show ip route - copy r s


4) 플래쉬 확인

enable - show flash - copy r s

HostName 수정 예시


Router1> en

Router# conf t

Router1(config)# hostname ICQA

ICQA(config)# exit

ICQA(config)# copy r s

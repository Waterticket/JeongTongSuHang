1. IP 주소 설정



이 부분은 딱히 어렵지 않게 풀 수 있다.

단, 아래와 같이 비트수로 나올때만 주의해주면 된다

IP Address : 10101100.00010000.10010110.01110011

Subnet Mask : 22 bit



2. DHCP 주소 설정



IPv4 에서 오른쪽 마우스 클릭 > [새 범위(P)] 클릭 후 제시문 대로 설정



3. 웹 서버 (IIS) 설정



사이트 제작시 [사이트] 에서 오른쪽 마우스 클릭 > [웹 사이트 추가] 클릭 후 제시문 대로 설정





웹 사이트는 위 사진과 같이 설정하면 된다. (난이도: 하)



4. 도메인 네임 서버 (DNS) 설정



문제가 위와 같을때 제시문 대로 설정하면 된다.





정방향에 경우 [정방향 조회 영역] 에서 오른쪽 마우스 클릭 > [새 영역(Z)] 클릭 후 제시문 대로 설정

주의. 정방향(A,CNAME 등..), 역방향(PTR 등..) 구분 잘하기!





영역에 들어가 마우스 오른쪽 클릭 > [새 호스트..] 혹은 [새 별칭..] 클릭해서 세부설정 (난이도: 하)
